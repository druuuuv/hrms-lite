HRMS Lite: Full-Stack Employee & Attendance Portal
A streamlined, production-ready Human Resource Management System built to handle real-time employee tracking with data-driven insights.

Live Link
Live Application: https://hrms-lite-fixed.onrender.com/

GitHub Repository: https://github.com/druuuuv/hrms-lite

Key Features
Real-time Dashboard: Instant visual feedback with color-coded status indicators (Green/Red) for daily attendance.

Attendance History: Dedicated historical log for every employee, allowing admins to track long-term patterns.

Administrative Control: Full CRUD capabilities, including the ability to retroactively edit past attendance records via a secure interface.

Smart Validations:

Gmail-Exclusive: Enforces corporate-standard email formatting using Regex.

Numeric Data Integrity: Prevents non-numeric input for Employee IDs to ensure database consistency.



Tech Stack:
Backend: Django (Python) - RESTful API architecture
Frontend: React & Tailwind CSS - Responsive, single-page application"
Database: SQLite - Optimized for efficient relational data storage
